🧠 OOP Principles in This App
🔒 Encapsulation
All attributes are private. Access only via getters/setters. UI never touches raw data directly.
🧬 Inheritance
StudySession (abstract) → AcademicSession / SkillSession → 4 child classes, sharing common behavior.
🎭 Polymorphism
Each child class overrides getMotivation() with unique messages. Timer calls the same method, different results per class.
🌫️ Abstraction
StudySession is abstract — forces child classes to implement getSessionType(), getSummary(), getMotivation().
