🧠 OOP Pillars Applied
🔒 Encapsulation
All attributes are private. Data is only accessed through public getters and setters. The UI never touches raw fields directly.
🧬 Inheritance
StudySession (abstract) → AcademicSession / SkillSession (parents) → 4 child classes. Each level inherits and extends the one above.
🎭 Polymorphism
Every child class overrides getMotivation(). The timer calls the same method on any session object — but gets a different result based on its actual class.
🌫️ Abstraction
StudySession is abstract and cannot be instantiated. It enforces a contract: all subclasses MUST implement getSessionType(), getSummary(), and getMotivation().
